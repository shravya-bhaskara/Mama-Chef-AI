# MamaChef AI 🍳

**MamaChef AI** is an AI-powered cooking assistant designed to simplify everyday meal decisions. The app helps users quickly discover recipes using ingredients they already have, generate weekly meal plans, and explore healthy and culturally relevant cooking ideas.

The goal of MamaChef AI is to reduce the daily stress of deciding *“What should I cook today?”* by providing intelligent, fast, and personalized recipe suggestions.

## Key Features

* **Ingredient-Based Recipes**
  Enter ingredients available at home and instantly generate recipes.

* **AI Weekly Meal Planner**
  Automatically create a 7-day meal plan including breakfast, lunch, dinner, and snacks.

* **Health-Oriented Meal Planning**
  Generate balanced meal plans based on nutrition goals such as calories, protein, and vitamins.

* **5-Minute Dinner Rescue**
  Quick meal ideas for busy days using minimal ingredients and preparation time.

* **Festival & Cultural Recipes**
  Discover recipes based on festivals, regional cuisine, and cultural traditions.

* **Recipe Packs (Premium)**
  Curated collections like lunchbox meals, quick family dinners, and healthy snack recipes.

## Tech Stack

* Frontend: Web application (React / modern JS stack)
* Backend: API-based architecture
* AI Models: Large Language Models for recipe and meal generation
* Deployment: Replit / Cloud deployment platforms
* Database: User data, preferences, and saved meal plans

## Vision

MamaChef AI aims to become a personalized cooking companion that adapts to each user's lifestyle, dietary needs, and cultural preferences—making home cooking easier, faster, and more enjoyable.
